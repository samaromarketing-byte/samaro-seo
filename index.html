How to deploy this tool to seo.samaro.ai
1

Download this file

Right-click anywhere on the page → Save as → save as index.html. Or click the button below.

2

Create a GitHub repo

Go to github.com → New repository → name it samaro-seo → Private → Create. Then upload your index.html to the root.

3

Deploy on Vercel

Go to vercel.com → Sign up with GitHub → Add New Project → select samaro-seo → Deploy. Live in 60 seconds at samaro-seo.vercel.app.

4

Add custom domain seo.samaro.ai

Vercel → your project → Settings → Domains → add seo.samaro.ai. Add this DNS record at your registrar:

Type: CNAME Name: seo Value: cname.vercel-dns.com
5

Update in future

Edit index.html on GitHub directly (pencil icon) → Commit changes. Vercel auto-deploys in 30 seconds.
