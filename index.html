How to deploy this tool to your own domain
1

Download this file

This page is the deployable file — just save it. Right-click anywhere on the page in your browser → Save as → save as index.html. Or use the button below.

2

Create a free GitHub account & repo

Go to github.com → Sign up (free) → Click New repository → Name it samaro-seo → Set to Private → Click Create repository.

3

Upload your index.html to GitHub

Inside your new repo, click uploading an existing file → drag your index.html onto the page → scroll down → click Commit changes. No coding required.

4

Deploy on Vercel (free)

Go to vercel.com → Click Sign Up → choose Continue with GitHub → authorise Vercel.

Then click Add New Project → select your samaro-seo repo → click Deploy.

Vercel auto-detects the HTML file. Your app is live at samaro-seo.vercel.app within 60 seconds.

5

Add your custom domain (e.g. seo.samaro.ai)

In Vercel → open your project → Settings → Domains → type seo.samaro.ai → click Add.

Vercel shows you a CNAME record to add. Go to your domain registrar (wherever samaro.ai DNS is managed — likely Cloudflare, GoDaddy, or Namecheap).

Add this DNS record:

Type: CNAME Name: seo Value: cname.vercel-dns.com

Wait 5–10 minutes → your tool is live at seo.samaro.ai.

6

Password protect it (optional)

To keep it private, create a vercel.json file in your repo alongside index.html with this content:

{ "passwordProtection": { "password": "yourpassword123" } }

Vercel will prompt for the password before showing the page. (Requires Vercel Pro — $20/mo — for password protection. Free alternative: add a simple JS prompt at the top of index.html.)

7

Update the tool in future

Any time you want to update the tool (add tasks, change keywords etc.), edit index.html on GitHub directly → click the pencil icon → make changes → Commit changes. Vercel auto-deploys within 30 seconds. No manual re-upload needed.
